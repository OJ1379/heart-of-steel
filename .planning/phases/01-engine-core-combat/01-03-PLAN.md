**`drawHealthBar(ctx, x, y, w, h, current, max, isPlayer)` function:**
Exact layout per UI-SPEC:
- Border: `ctx.fillStyle = '#000000'; ctx.fillRect(x - 1, y - 1, w + 2, h + 2);`
- Empty background: `ctx.fillStyle = '#333333'; ctx.fillRect(x, y, w, h);`
- Calculate percentage: `pct = current / max`
- Color: `pct > 0.5 ? '#00CC00' : pct > 0.25 ? '#FFCC00' : '#CC0000'`
- Fill width: `fillW = Math.floor(w * pct)`
- Player (isPlayer=true): fill from left: `ctx.fillRect(x, y, fillW, h)`
- CPU (isPlayer=false): fill from right: `ctx.fillRect(x + w - fillW, y, fillW, h)`
- Highlight: `ctx.fillStyle = 'rgba(255,255,255,0.3)'; ctx.fillRect(x, y, fillW > 0 ? (isPlayer ? fillW : w) : 0, 1);` -- adjust for right-fill on CPU side

**`drawRoundPips(ctx, x, y, winsCount)` function:**
- Draw 2 small circles (approximated as 4x4 filled rectangles per UI-SPEC):
  - Pip 1 at (x, y): filled `#FFCC00` if `winsCount >= 1`, else `#555555`
  - Pip 2 at (x + 8, y): filled `#FFCC00` if `winsCount >= 2`, else `#555555`
- Each pip: 4x4 fillRect

**`drawHUD(ctx, player, cpu, roundTimer, p1Wins, p2Wins)` function:**
All positions from UI-SPEC HUD Layout Contract:
1. P1 name "TRUMP": `drawText(ctx, player.def.name.toUpperCase(), 16, 8, '#FFFFFF', 8, 'left')` (with 1px black shadow)
2. P1 health bar: `drawHealthBar(ctx, 48, 8, 128, 8, player.health, player.def.stats.health, true)`
3. Timer label "TIME": `drawText(ctx, 'TIME', 200, 2, '#FFFFFF', 8, 'center')`
4. Timer value: `drawText(ctx, String(Math.ceil(roundTimer / 60)), 200, 12, timerColor, 8, 'center')` where `timerColor = roundTimer < 600 ? '#CC0000' : '#FFFFFF'` (red under 10 seconds = 600 frames)
5. P2 health bar: `drawHealthBar(ctx, 224, 8, 128, 8, cpu.health, cpu.def.stats.health, false)`
6. P2 name "CPU": `drawText(ctx, cpu.def.name.toUpperCase(), 360, 8, '#FFFFFF', 8, 'right')` -- note: for right-aligned text, set `ctx.textAlign = 'right'` and draw at x position
7. P1 round pips: `drawRoundPips(ctx, 120, 20, p1Wins)`
8. P2 round pips: `drawRoundPips(ctx, 268, 20, p2Wins)`

**Round state variables (add to fight scene):**
```javascript
let roundTimer = 3600;   // 60 seconds * 60 ticks per second
let p1Wins = 0;
let p2Wins = 0;
let currentRound = 1;
let koSplashTimer = 0;   // counts down from 90 when KO occurs
let koTriggered = false;
let roundWinner = null;   // 'player' or 'cpu'
```

**Wire into fight scene update:**
- Each tick (when not in hit stop): decrement `roundTimer`
- If `roundTimer <= 0` (time out): determine winner by higher health percentage, trigger KO sequence
- If fighter health <= 0: set `koTriggered = true`, `koSplashTimer = 90`, `roundWinner = ...`
- While `koTriggered && koSplashTimer > 0`: decrement `koSplashTimer`, skip fighter updates, render KO splash
- When `koSplashTimer` reaches 0: increment winner's win count, check for match end (2 wins = match over -> victory scene), else reset round (reset health, positions, timer, increment `currentRound`)

**Wire into fight scene render:**
After drawing stage and fighters:
```javascript
drawHUD(ctx, player, cpu, roundTimer, p1Wins, p2Wins);
```

During KO splash (`koTriggered && koSplashTimer > 0`):
- Draw "K.O.!" centered on screen: `drawText(ctx, 'K.O.!', 200, 110, '#FFFFFF', 16, 'center')` -- 16px bold monospace per UI-SPEC copywriting
- Flash effect: alternate text color between `#FFFFFF` and `#CC0000` every 5 frames

**Update victory scene:**
Victory scene should display:
- Dark background `#0A0A1A`
- Winner name in large text: e.g., "TRUMP WINS!" or "CPU WINS!" at y=100, centered, 16px
- "PRESS ENTER TO CONTINUE" blinking at y=190
- On Enter: reset all match state (p1Wins, p2Wins, currentRound) and transition to title

**Fight scene enter() reset:**
When entering fight scene, reset: both fighter health to full, positions to starting (player x=100, cpu x=300), roundTimer = 3600, koTriggered = false, both fighters to idle state. Keep p1Wins/p2Wins across rounds within a match, reset on new match from title.
