```javascript
// Cinematic zoom during wall break (STG-05)
let wallBreakZooming = wallBreakActive && wallBreakTimer > 0;
if (wallBreakZooming) {
  const progress = 1 - (wallBreakTimer / 30); // 0 -> 1 over 30 frames
  const zoomScale = 1.0 + progress * 0.5;     // 1.0x -> 1.5x
  const focusX = wallBreakSide === 'right' ? STAGE_RIGHT : STAGE_LEFT;
  const focusY = GROUND_Y - 20;
  ctx.save();
  ctx.translate(focusX, focusY);
  ctx.scale(zoomScale, zoomScale);
  ctx.translate(-focusX, -focusY);
}
```

2. AFTER drawing fighters but BEFORE drawing HUD (between line 4036 and 4039), close the zoom transform:
```javascript
if (wallBreakZooming) {
  ctx.restore();
}
```

- Stage background and fighters are drawn inside the zoom transform (zoomed in)
- HUD is drawn OUTSIDE the zoom transform (stays at 1:1 scale, not affected)
- Round splash, KO splash, match over overlays render at 1:1 (outside zoom)
- Ultimate animations render inside zoom (they overlay the fight area)
